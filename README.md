# Serhii Kovaliyk - Personal Portfolio 🚀

Welcome to the repository of my personal portfolio website! I am a Computer Science student passionate about software development, AI, and building engaging user interfaces.

## 🌟 About

This portfolio is a showcase of my journey in tech, my skills, and the projects I am currently working on. It's built with modern web technologies, focusing on clean design, performance, and responsive layout.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, JavaScript
- **Styling**: Tailwind CSS for rapid and responsive UI development
- **Typography**: Inter & JetBrains Mono fonts
- **Animations**: Custom CSS keyframes (Fade-up, Slide-in, Infinite Marquee)

## ✨ Features

- **Responsive Design**: Fully adaptable layout for mobile, tablet, and desktop devices.
- **Dark Mode Aesthetic**: A sleek, modern dark theme with subtle grid background accents and glassmorphism elements.
- **Interactive UI**: Micro-animations, smooth scrolling, and hover effects for a premium feel.
- **Bento Grid Layout**: Trendy and structured presentation of skills and focus areas.
- **Marquee Animation**: Seamless infinite scrolling banner showcasing core technologies.

## 🚀 Running Locally

To view this project locally, you don't need any complex build tools. Just follow these simple steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/KovaliykSerhii/Web_Portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Web_Portfolio
   ```
3. Open `index.html` in your favorite browser.
   *Tip: Use an extension like Live Server in VS Code for the best development experience with hot reloading.*
